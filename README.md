# 🐾 Animal Shelter Management System

A simple Java console application that simulates the basic operations of an animal shelter. The project demonstrates Object-Oriented Programming (OOP) concepts by managing animals, adopters, vaccinations, and the adoption process.

---

## 📌 Project Overview

This application allows an animal shelter to register animals and adopters, match adopters based on their preferences, vaccinate animals before adoption, and generate a summary report of shelter activities.

The project was developed to practice Java OOP concepts such as abstraction, inheritance, encapsulation, polymorphism, and class relationships.

---

## ✨ Features

- Register animals into the shelter
- Register adopters
- Match adopters with suitable animals
- Vaccinate animals before adoption
- Complete the adoption process
- Display shelter activity report
- Supports different animal types (Dog and Cat)

2. Open the project in Eclipse.

3. Navigate to:


4. Run the `main()` method.
